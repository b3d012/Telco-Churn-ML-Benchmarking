01. Problem

Why this benchmark matters

Telecom churn prediction is valuable because companies want to identify customers who are likely to leave and prioritize retention outreach before revenue disappears. This project focuses on choosing and interpreting the right machine learning model, not just cleaning the dataset.

02. Project Positioning

10. Model Interpretation

Permutation importance is the main explanation method

Top churn drivers from permutation importance are Contract, Partner, OnlineBackup, Dependents, tenure, SeniorCitizen, TotalCharges, StreamingMovies, MonthlyCharges, and MultipleLines. Feature importance should be read as predictive signal, not causal proof.

SHAP did not generate a figure in this repo, so the project falls back to permutation importance for the reliable explanation layer.
